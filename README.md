# Filters-in-C
Applies image processing in C.

## Available filters -

- BLUR
- SEPIA
- REFLECT
- GRAYSCALE

## How to run it?

- Compile the source files -
```
make filter
```
- Run the binary file (Syntax - ./filter -flag Infile.bmp Outfile.bmp). Example for blurring an image -
```
./filter -b images/yard.bmp images/bulredyard.bmp
```
